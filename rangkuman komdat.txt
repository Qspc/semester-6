- arsitektur jaringan = rancangan yg menyatakan bagaimana jaringan dibangun dan dioperasikan
- jaringan komunikasi = sekumpulan perangkat, mekanisme, prosedur, dan fasilitas yg menyediakan sebuah layanan
- node jaringan = mesin/komputer yg menyediakan koneksi bagi user (circuit switching) dan merelay data user (komdat)
- end system = jaringan memberikan pelayanan yaitu memindahkan informasi antar pengguna yg berlokasi di berbagai tempat

- circuit switching = jalur komunikasi khusus yg dibangun selama durasi komunikasi, c: telepon
- paket switching = data terkirim dalam paket kemudian melintas antar node. c: sms

- layer = pembagian fungsi kedalam grup yg dapat dikontrol oleh sebuah protokol
- protokol = himpunan aturan yg memandu interaksi antar entitas komunikasi

- 7 OSI layers :
1. application layer = menyediakan layanan untuk program aplikasi pengguna. c: html, dns, smtp
2. presentasion layer = layanan penyesuaian format data agar dapat dibaca. C: ascii, jpg
3. session layer = mengontrol trasnfer data antar terminal
4. transport layer = proses transfer data dari mesin awal ke mesin tujuan. gunain port number
5. network layer = transfer packet melalui berbagai link jaringan. pake IP
6. data link layer = transfer frame melalui koneksi langsung. kelompokan bit kedalam frame. c: hdlc
7. physical layer = tranfer bit melalui link, c: ethernet

layer presentasion = format data digital

line coding = merubah data biner menjadi informasi digital

error control = digunakan ketika sustem transmisi tak dapat memenuhi kebutuhan aplikasi

polynomial codes = merepresentasikan nilai biner kedalam polinom
